Clarke's Model

This model is based on statistical properties rather than direct simulation. It assumes isotropic scattering, meaning signals arrive from all directions with equal probability, leading to a theoretical Bessel function for autocorrelation and a classic "bathtub" shape for the Doppler power spectrum.

  • Scatterers are uniformly distributed around the receiver.
  • Each signal path has equal average power.

Realistic Model Implementation

This model simulates a practical Rayleigh fading channel using Clarke's sum of sinusoids approach. The implementation works as follows:

  • Channel Generation: We generate a complex baseband channel response h(t) = hI(t) + jhQ(t) using Clarke's sum of sinusoids model with N multipath components.
  • Signal Transmission: We transmit a simple unmodulated carrier signal s(t) = 1 (unit amplitude at baseband).
  • Received Signal: The received signal is r(t) = h(t) ⊗ s(t) = h(t), so the envelope |r(t)| = |h(t)| follows a Rayleigh distribution.
  • Analysis: We compute the autocorrelation R(τ) and power spectral density S(f) of the received signal r(t) and compare them with Clarke's theoretical curves.
